gitbook的使用

gitbook简介

gitbook 是本书的编辑工具，简单易用。

下载和安装

  • 下载 typora (一款编辑工具)
  • npm install -g gitbook-cli
  • gitbook init
  • gitbook build ./ book
  • gitbook serve --port 8888

不要只用git build，github.io page 不识别下滑线。除非把根目录的.nojekyll放入下滑线目录，不过太麻烦。 win7创建命令：type nul> .nojekyll

位置匹配

  1. 什么是位置？

    相邻字符之间的位置

  2. 如何匹配位置

    在ES5中，共有6个锚字符：

    ^ $ \b \B (?=p) (?!p)

    1. 脱字符和美元符号

      ^（脱字符）匹配开头，在多行匹配中匹配行开头。

      $（美元符号）匹配结尾，在多行匹配中匹配行结尾。

      练习1: "hello".replace(/^|$/g, '#')

      answer

      练习2: "I\nlove\njavascript".replace(/^|$/gm, '#')

      answer

    2. \b和\B

      \b 代表什么?

      answer
      var result = "[JS] Lesson_01.mp4".replace(/\b/g, '#');

console.log(result);
      answer
      var result = "[JS] Lesson_01.mp4".replace(/\B/g, '#');

console.log(result);
      answer

    3. (?=p)和(?!p)

      匹配 : p前面的那个位置

      var result = "hello".replace(/(?=l)/g, '#');

console.log(result);
      answer
      var result = "hello".replace(/(?!l)/g, '#');

console.log(result);
      answer
